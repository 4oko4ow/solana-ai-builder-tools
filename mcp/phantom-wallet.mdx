# Phantom MCP Server (Wallet Actions)

**What it is:** An MCP (Model Context Protocol) server that enables MCP-compatible clients (agents) to perform Phantom wallet actions across Phantom-supported chains (e.g. sign, swap, manage addresses).  
Status: **Preview**.

## Why it matters
- Lets an AI agent interact with Phantom in a tool-driven way
- Useful for automation, trading UX experiments, and agentic workflows

## Install
```bash
npm i @phantom/mcp-server
